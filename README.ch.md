# 🗿 聖經遊戲 2025

介紹
---
這款遊戲以《聖經》中的《出埃及記》為背景，共分為三個主要關卡：十災篇、紅海篇和十誡篇。

在前兩個關卡中，主角摩西會被怪物追趕。玩家可使用 W、A、S、D 鍵進行移動，並可射擊敵人進行攻擊。遊戲中會出現各種障礙物阻礙摩西前進。
要通過第一與第二關，玩家必須抵達畫面右下角的出口，才能進入下一關。

第三關（十誡篇）中，目標是收集所有 十塊石板。當玩家收集完成後，才能成功過關！
